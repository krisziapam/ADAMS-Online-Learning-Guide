# ADAMS Online Learning Guide

GitHub Pages-ready static website for the ADAMS database project reviewer.

## Features

- W3Schools-style lesson sidebar
- Global search for lessons, SQL, ERD/EERD, Q&A, and full uploaded reviewer text
- SQL TryIt editor using safe offline ADAMS sample tables
- Interactive ERD/EERD with clickable tables
- Normalization simulator
- Cardinality trainer
- SQL short assessment
- Flashcards
- Mock oral defense
- Original PDF viewer links

## Run locally

Open `index.html` in any modern browser.

## Deploy on GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder.
3. Go to **Settings > Pages**.
4. Choose **Deploy from branch**.
5. Select the `main` branch and `/root` folder.
6. Open your GitHub Pages URL.

No backend, no database connection, and no build step required.
