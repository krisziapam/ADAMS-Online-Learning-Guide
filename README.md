# ADAMS Online Learning Guide - Improved Final

GitHub Pages ready static website.

## Included
- W3Schools-style ADAMS reviewer
- Per-lesson Activity Center
- ADAMS Database Explorer
- SQL Test Lab with offline sample database modeled from ADAMS schema
- Normalization Simulator UNF → 1NF → 2NF → 3NF
- ERD/EERD guide, flashcards, quiz, PDF source viewer

## Deploy
Upload all files in this ZIP to the root of your GitHub repository and enable GitHub Pages from the main branch.

Note: GitHub Pages is static. The SQL Test Lab is a simulator and does not connect to MySQL. Use MySQL Workbench for real execution.
