# ADAMS Online Learning Guide

This is a complete static learning portal for the ADAMS database project. It can be deployed directly on GitHub Pages.

## Features

- Sidebar lesson navigation
- ADAMS Reviewer Center for Panel Q&A, SQL Codes, Normalization, Mixed Review, Flashcards, and Mock Defense
- Search across lessons, SQL, ERD/EERD, Q&A, and source reviewer text
- SQL TryIt-style offline simulator using sample ADAMS tables
- Interactive ERD/EERD visual guide
- Dark mode toggle

## Deployment

1. Upload all files to your GitHub repository.
2. Go to Settings > Pages.
3. Choose Deploy from branch.
4. Select the main branch and root folder.
5. Open the generated GitHub Pages URL.
